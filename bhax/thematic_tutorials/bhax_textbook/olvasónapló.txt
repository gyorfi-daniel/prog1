A C programozási nyelv


1.1 Indulás 

A könyv alapszemléte, hogy a programozást egyféleképpen lehet megtanulni, ez pedig az, ha írjuk a programokat. Ehhez hűen kezdjük el C tanulmányainkat is, már az első oldalon szerepel az első kódunk.

main ()
{
printf ("Figyelem, emberek! \ n");
}


Itt megismerkedhetünk az egyik legfontosabb fogalommal, a Függvénnyel. 
-Minden program tartalmaz egy vagy több függvényt.
-Neve tetszőleges, de a program futása a maintől kezdődik. Itt hívjuk meg a használni kívánt többi függvényt.
-Függvények közti adatátvitelt az argumentumok használata segíti.

1.2 Változók és aritmetika

Comment:
-Program rövid leírása, annak megértését segítő szöveg
-Fordító figyelmen kívül hagyja
-jelölése /* ide megy a comment */ vagy //ide az egy soros comment

Változók:
-int egész
-float lebegőpontos szám
-char karakter
-short rövid egész
-long hosszú egész
-double dupla pontosságú lebegőpontos szám

Értékadó utasítás : pl.: lower = 0;

while ciklus :
-megvizsgáljuk hogy igaz-e a feltétel
	-ha igen akkor végrehajtjuk a ciklustörzset
	-ha nem akkor nem hajtjuk végre.
-feltétel () között, törzs {} között (egy soros ciklustörzsnél elhagyható)
ciklustörzset beljebb írjuk az átláthatóság érdekében

Printf( "kinyomtatandó karakterlánc" , kinyomtatandó változók )
-%d decimális
-%c karakter
-%s karakterlánc ...
